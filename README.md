> **EcoKids India** transforms dry environmental education into an addictive, points-driven adventure. Students earn Eco-Points for planting trees, sorting waste, conserving water, and acing quizzes тАФ all verified by teachers and tracked in real-time across school, district, and state leaderboards. Every feature adapts to the child's grade band, speaks their mother tongue, and works offline when the Wi-Fi disappears.

## тЬи Features

### ЁЯОо Gamification Engine
- **Eco-Points & XP** тАФ Every quiz, experiment, and real-world activity earns XP with streak multipliers
- **Badges & Achievements** тАФ Unlock "Tree Guardian", "Water Warrior", "Climate Hero" and 40+ badges
- **Leaderboards** тАФ Live rankings at student тЖТ class тЖТ school тЖТ district тЖТ state level
- **Confetti Celebrations** тАФ canvas-confetti powered reward moments on quiz pass

### ЁЯХ╣я╕П Interactive Games
| Game | Topic | Mechanic |
|------|-------|----------|
| Waste Sort Rush | Waste Management | Drag-and-drop sorting |
| Water Journey | Water Conservation | Resource flow puzzle |
| Tree Planter | Biodiversity | Click-to-grow simulation |
| Climate Simulator | Climate Change | Cause-effect strategy |

### ЁЯФм Hands-on Experiments
- Step-by-step guided science experiments (Water Quality, Waste Segregation, Solar Energy, and more)
- **Photo proof submission** with camera or file upload
- **Teacher approval workflow** тАФ verified submissions release Eco-Points instantly
- Immutable audit log entry created for every review action

### ЁЯУЪ Grade-Adaptive Learning
The UI auto-detects the student's band and transforms the entire experience:

| Band | Grades | Style |
|------|--------|-------|
| ЁЯМ▒ Seedling | 1тАУ3 | Big fonts, cartoon illustrations, voice-friendly |
| ЁЯФн Explorer | 4тАУ6 | Structured cards, progress rings |
| тЪб Challenger | 7тАУ9 | Dense data, streak counters, timed quizzes |
| ЁЯОУ Expert | 10тАУ12 | Assessment-grade UI, percentile scores |

### ЁЯПл School Infrastructure
- **QR Code Login** тАФ printable A4 QR cards for instant classroom login (no password needed)
- **Multi-tenant Isolation** тАФ school data is cryptographically isolated; cross-school queries are blocked and audit-logged
- **Parental Consent System** тАФ OTP-verified digital consent with DPDP Act 2023 compliant version history
- **Offline Activity Submission** тАФ IndexedDB queue syncs automatically on reconnect
- **Principal Reports** тАФ PDF/CSV management reports with 5-slide presentation mode

### ЁЯСйтАНЁЯПл Teacher & Admin Tooling
- Submission review queue with approve/reject + rich feedback
- Class progress heatmaps and engagement analytics
- Bulk parental consent send/reminder/export
- Immutable audit logs тАФ every sensitive action is recorded with actor, IP, user-agent, and compliance flags

## ЁЯЪА Quick Start

### Prerequisites
- **Node.js 18+**
- **MongoDB** (local) or a free [MongoDB Atlas](https://mongodb.com/atlas) cluster
- **npm 9+**

### 1 тАФ Clone

```bash
git clone https://github.com/itskum47/Ecokids-India.git
cd Ecokids-India
```

### 2 тАФ Configure environment

```bash
# server/.env
MONGODB_URI=mongodb://localhost:27017/ecokids-india
JWT_SECRET=change-me-to-a-long-random-secret
PORT=5001
NODE_ENV=development
CLOUDINARY_CLOUD_NAME=your_cloud
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
```

### 3 тАФ Install dependencies

```bash
# From repo root тАФ installs both server + client
npm install
cd server && npm install
cd ../client && npm install
```

### 4 тАФ Seed demo data *(optional but recommended)*

```bash
cd server && npm run seed
```

### 5 тАФ Run

```bash
# Terminal A тАФ Backend
cd server && npm start

# Terminal B тАФ Frontend
cd client && npm run dev
```

Open **http://localhost:5173** ЁЯОЙ

### 6 тАФ Or use Docker

```bash
docker-compose up --build
```

## ЁЯРЫ Troubleshooting

<details>
<summary><strong>MongoDB connection refused</strong></summary>

Start MongoDB locally: `mongod` тАФ or swap `MONGODB_URI` for a MongoDB Atlas URI.
</details>

<details>
<summary><strong>Port 5001 already in use</strong></summary>

```bash
lsof -ti:5001 | xargs kill -9
```
</details>

<details>
<summary><strong>Module not found / post-install errors</strong></summary>

```bash
rm -rf node_modules package-lock.json && npm install
```
</details>

<details>
<summary><strong>Translations not showing</strong></summary>

Locale files live at `client/src/i18n/locales/<lang>.json`. All 10 files are parity-checked at 550 keys each. If a key is missing, `i18next` falls back to `en.json`.
</details>
