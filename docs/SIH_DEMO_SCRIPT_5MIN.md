# SIH Demo Script (3-5 Minutes)

## Assigned Roles
- Demo driver: teacher1@dps-del.ecokids.in
- Backup presenter: admin@dps-del.ecokids.in

## Strong Opening (First 10 Seconds)
- Show current rank on leaderboard
- Show impact card totals
- Show current user points and streak

Line:
"We convert climate awareness into verified daily action, measurable impact, and real competition in schools."

## Time-Boxed Flow

### 0:00-0:30
- Student dashboard and hook
- Show rank + impact + streak

### 0:30-1:30
- Submit one activity
- Show status progression

### 1:30-2:30
- Teacher verifies submission
- Show approved state and trust metadata

### 2:30-3:30
- Return to student side
- Show reward timeline update and rank delta

### 3:30-4:30
- Show school/district impact summary and behavior metrics

### 4:30-5:00
- Failsafe proof: explain smoke/latency scripts and QR backup navigation

## Before/After Contrast Cues
- Before verify: pending status + old rank
- After verify: approved status + points increment + rank delta + impact delta

## No-Silence Rule
Always narrate each wait state:
- "Submission received, now entering verification queue."
- "Verification complete, reward and ranking pipeline triggered."
- "Impact totals recomputed and visible now."

## Wow Moment
- Rank jump + impact delta visible within one cycle after teacher verification.
